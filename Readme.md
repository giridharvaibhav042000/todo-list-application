#  Todo List Application

A full-stack **Todo List Application** built using **React**, **Node.js**, **Express**, and **MongoDB** with **JWT authentication**.  
Users can register, log in, and manage their own tasks with search, sort, and update features.

## Features

- JWT Authentication (Login & Register)
- User-specific Todos
- Add / Update / Delete tasks
- Search todos by title
- Sort todos by name and status
- Protected Routes
- Responsive UI
- Cloud deployment (Vercel)
